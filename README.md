## 👊 AI knows All

🏏 V1.0 (2026년 05월 09일)

● 본 게임은 **2D 그림 그리기 게임**으로, CNN 모델이 손그림을 인식하는 것을 유니티 상에서 구현한 게임입니다.

● PyTorch로 CNN 모델을 구현해 ONNX Export후, 유니티 Sentis와 호환하여 실시간으로 AI가 사용자가 그린 손그림을 49개의 클래스 중 예측한 결과값을 출력하는 기술적인 프로젝트 입니다.
